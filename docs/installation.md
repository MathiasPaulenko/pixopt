# Installation

optimg requires Python 3.9 or newer.

## From PyPI

```bash
pip install optimg
```

## With HEIC/HEIF support

```bash
pip install optimg[pillow-heif]
```

Or install the extra directly:

```bash
pip install pillow-heif
```

## From source

```bash
git clone https://github.com/yourusername/optimg.git
cd optimg
pip install -e ".[dev]"
```

## Verify installation

```bash
optimg --help
```
