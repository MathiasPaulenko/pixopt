# Installation

pixopt requires Python 3.9 or newer.

## From PyPI

```bash
pip install pixopt
```

## With HEIC/HEIF support

```bash
pip install pixopt[pillow-heif]
```

Or install the extra directly:

```bash
pip install pillow-heif
```

## From source

```bash
git clone https://github.com/MathiasPaulenko/pixopt.git
cd pixopt
pip install -e ".[dev]"
```

## Verify installation

```bash
pixopt --help
```
